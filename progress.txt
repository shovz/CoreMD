- US-006 prompt engineering trap (2026-04-25): including literal example values in an OpenAI JSON template (e.g. `"correct_option": 0` for Q1, `1` for Q2, `2` for Q3) causes the model to copy those values verbatim every run. Always shuffle options programmatically after parsing the response — never rely on the model to randomize positions.
